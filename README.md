# Craig's Navigator

Dark-mode, privacy-first, autistic-friendly life companion and acoustic review system.

## Current Alpha Features

- Morning report
- Notification + humour engine
- Voice companion input
- Reality monitor
- Waveform visualisation
- Overnight event logging
- Clip capture + replay scaffold
- Companion sharing approval
- Calm evidence-first reasoning

## Local Run

npm install
npm run dev

## Build

npm run build

## Firebase Hosting

firebase login
firebase init hosting
firebase deploy

## Safety

Craig's Navigator reviews acoustic evidence and wellbeing context.
It does not diagnose, determine reality or replace clinical support.
Enhanced audio is probabilistic and should be compared with raw recordings.
